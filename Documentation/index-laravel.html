Installation Process

Inorder to run laravel we need to install composer by the following command

Note : Install composer through command prompt.

composer install

After installation run following command to get laravel development server

php artisan serve

FTP Uploading Process

Step 1:

Create a folder in ftp home directory(e.g laravel-project)

Step 2:

Paste all your project content in laravel-project except public folder

Step 3:

Place public folder content in /home/public_html

Step 4:

The last important change on your code is in /home/public_html/index.php , you should change
require __DIR__.'/../bootstrap/autoload.php';
$app = require_once __DIR__.'/../bootstrap/app.php';


These two lines to
require __DIR__.'/../../laravel-project/bootstrap/autoload.php';
$app = require_once __DIR__.'/../../laravel-project/bootstrap/app.php';

How to Upgrade Larvel Version

If you want Upgrade laravel version, So Please to follow the below steps, otherwise leave this section

Step 1:

Replace below lines in package.json file which is placed in root directory

"php": "^7.2.5",
"fideloper/proxy": "^4.2",
"fruitcake/laravel-cors": "^1.0",
"guzzlehttp/guzzle": "^6.3",
"laravel/framework": "^7.0",
"laravel/tinker": "^2.0"

Step 2:

Delete vendor folder

Step 3:

Run this following command in command prompt(give root directory path)

Composer update
